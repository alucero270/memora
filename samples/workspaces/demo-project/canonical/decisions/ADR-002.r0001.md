## Context
The team explored whether semantic retrieval should be added to Memora to help
with long-term memory, large-project backfill, and fuzzy discovery.

## Decision
Memora core remains deterministic, explainable, and non-semantic. Semantic
retrieval is out of scope for core Memora. Any future exploration must stay
outside the core truth model and must not replace deterministic final context
assembly.

## Alternatives Considered
- add semantic retrieval directly to core ranking
- make embedding-backed retrieval the default context path
- avoid future-track discussion entirely

## Consequences
Core retrieval boundaries remain crisp. Future ideas can still be explored, but
only as deferred or advisory work that does not bypass approval-governed
project memory.

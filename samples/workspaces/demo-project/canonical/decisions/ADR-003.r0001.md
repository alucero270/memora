## Context
The team explored whether Memora's MVP should be monetized immediately or
whether the strongest first offering is a fully usable local-first core.

## Decision
The local-first MVP is free. Paid tiers begin where Memora adds convenience,
multi-machine polish, team workflow support, or scale-oriented assistance.

## Alternatives Considered
- gate core deterministic retrieval and proposal workflow behind a paid plan
- make the MVP mostly free but paywall the local agent integration path
- defer all packaging discussion until after broader feature expansion

## Consequences
Memora's core identity stays clear: trustworthy local project memory is the
baseline product. Monetization focuses on leverage and convenience rather than
on restricting filesystem-first truth and governance.

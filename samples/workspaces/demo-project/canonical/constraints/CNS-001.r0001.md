## Constraint
Memora's free tier must include the local canonical artifact model, lifecycle
rules, deterministic retrieval, rebuildable local index, and proposal-governed
local agent workflow.

## Why It Exists
If core truth and governance are paywalled, Memora stops feeling like reliable
project infrastructure and starts looking like gated storage.

## Implications
Paid layers should focus on sync, hosted convenience, team coordination,
backfill/import support, and advisory intelligence. They should not redefine
what counts as project truth.

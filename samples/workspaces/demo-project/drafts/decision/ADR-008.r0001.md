## Context
Issue #124 overlaps heavily with the more specific Milestone 10 issues for
Codex, ChatGPT, and golden-path workflow validation. Leaving it as a competing
implementation slice would blur milestone scope.

## Decision
Treat issue #124 as an umbrella or tracking issue for the broader attachment
goal, not as a separate implementation slice to execute in parallel with the
specific M10 issues.

## Alternatives Considered
- execute issue #124 directly alongside the M10 issue set
- close issue #124 immediately without preserving its tracking value
- split issue #124 into additional milestone issues before any M10 execution

## Consequences
The milestone can execute through the narrower issue set while still keeping a
high-level tracker for the overall external-client attachment goal.

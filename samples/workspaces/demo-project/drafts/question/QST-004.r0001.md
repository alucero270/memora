## Question
At what point should Memora require an explicit proposal-to-outcome link in the
external workflow instead of allowing looser outcome recording?

## Context
The Phase 2 draft keeps outcome recording in scope, but it also tries to avoid
over-coupling the first operational workflow before real usage proves what is
necessary.

## Possible Directions
- require linkage immediately in M10
- allow loose linkage in M10 and tighten later
- make linkage conditional by outcome type

## Resolution
Open. The current baseline leans toward looser linkage first, then revisiting
after real M10 usage.

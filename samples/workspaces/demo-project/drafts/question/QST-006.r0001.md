## Question
Should same-network sync between a mobile capture client and desktop Memora be
part of the first mobile milestone?

## Context
The user wants same-network sync eventually, but the current roadmap also tries
to protect the local-first MVP from prematurely taking on service discovery,
trust, and conflict-handling complexity.

## Possible Directions
- defer same-network sync entirely and use file transfer first
- design the sync path only and implement it later
- include a tiny same-network prototype only if it stays extremely small

## Resolution
Deferred. The first mobile slice should work without same-network sync.

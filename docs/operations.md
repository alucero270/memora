# Operations

Memora operations are centered on local filesystem truth and reviewable human
workflow.

Use [operator-workflows.md](operator-workflows.md) for the current local
operator path:

- project selection and artifact browsing
- approval queue review
- draft editing
- revision diff inspection
- rebuild and consistency troubleshooting

Operational docs should describe current behavior only. Future automation,
direct-write policies, and broader retrieval behavior belong in roadmap or
milestone docs until they are implemented.

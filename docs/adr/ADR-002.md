# ADR-002: Separation of Understanding and Retrieval

## Status
Accepted

## Context

Memora operates alongside Strata and other AI systems.

There is risk of overlapping responsibilities:
- retrieval (documents, passages)
- understanding (meaning, structure, traceability)

## Decision

Define strict boundary:

### Strata
- retrieves raw data
- indexes documents
- returns source material

### Memora
- owns structured artifacts
- defines relationships
- enforces lifecycle and validation
- assembles context
- produces understanding outputs

## Consequences

### Positive
- clear system responsibilities
- reduced architectural drift
- composability across systems

### Negative
- requires explicit ingestion boundaries
- may require transformation layer between systems

## Rules

- Memora never treats raw retrieval as truth
- Strata never constructs understanding
- all understanding must be represented as artifacts
